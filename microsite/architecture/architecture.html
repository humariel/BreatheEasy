Architecture

For Clients

Layer Description

Visual Layer

In the visualization layer we have the dashboards, both web and mobile application, that allow the user to interact with the system, check all relevant data and get notifications.

Service Layer

The service layer is responsible for providing services over the information provided by the data layer.

Data Layer

In this layer all the data provided by the gateway manager will be stored and transferred to the service layer.

Gateway Layer

This layer is composed by all the sensors that will provide all the data and sending it to the gateway.
